# アトランティスの深淵

「インカの黄金」をベースにした、よりスリリングなオンラインマルチプレイヤーボードゲーム。

## 遊び方

### ゲームの流れ
1. ルームを作成するか、ルームコードを入力して参加
2. ホストが「ゲーム開始」を押すとゲームスタート
3. 毎ターン「進む」か「帰る」を選択
4. 5ラウンド終了時に最も宝石を集めたプレイヤーの勝利

### 基本ルール
- **進む**: カードをめくって探索を続ける（宝石ゲット or 罠のリスク）
- **帰る**: 今持っている宝石を確定して安全に帰還
- **罠2枚**: 同じ罠が2枚出たら全員脱落、未確定宝石を失う

## 特徴

- 深海をイメージした美しいビジュアル
- 宝石分配時のダイナミックなアニメーション
- リアルタイムでのマルチプレイ同期
- スマートフォン対応のレスポンシブデザイン

## 技術スタック

- **フロントエンド**: React + TypeScript + Vite
- **スタイリング**: Tailwind CSS
- **バックエンド**: Firebase Realtime Database
- **ホスティング**: GitHub Pages

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
